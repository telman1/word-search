# Word Search MVP

A minimal working MVP for a word search + admin site using Strapi and Next.js.

## Project Structure
- `/backend-strapi` - Strapi CMS with PostgreSQL
- `/frontend-next` - Next.js React frontend

## Quick Start

### Backend (Strapi)
```bash
cd backend-strapi
docker-compose up
```

### Frontend (Next.js)
```bash
cd frontend-next
npm install
npm run dev
```

Access:
- Strapi Admin: http://localhost:1337/admin
- Next.js App: http://localhost:3000
